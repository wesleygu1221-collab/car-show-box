# Source Materials Index

## 原始 PDF

原始 PDF 当前保留在本地 `07_assets/source_materials/` 中，用作提案、deck、市场判断和规格核对的依据。

- `Feasibility Report.pdf`
  - 用途：可行性、Benelux 运营、Go / No-Go、运输、许可、保险、仓储、商业风险。
- `Viewbox _ Presentation (BRAND ACTIVATION).pdf`
  - 用途：品牌激活话术、Viewbox 基础卖点、非汽车案例、法语市场表达。
- `Viewbox _ Presentation (Car Showroom) .pdf`
  - 用途：汽车展厅场景、Porsche Panamera 案例、英文展示话术。
- `Viewbox+-+Catalogue+[2025].pdf`
  - 用途：规格、模块化能力、Kia / Porsche / HOKA / Booking.com / Coca-Cola 等案例。

## 使用规则

- 对外提案引用参数时，优先使用 catalogue 和 showroom presentation。
- 谈风险和落地时，优先使用 feasibility report。
- 谈创意和品牌体验时，优先使用 brand activation presentation。
- 涉及 OMODA & JAECOO 车型参数时，必须再核对当地官网或经销商资料。

## 可提取素材类型

- Viewbox 产品参数
- 汽车品牌案例
- 多层空间示意
- LED、玻璃、门、墙板、天花等模块信息
- Benelux 可行性风险清单
- 场地类型与运营要求
