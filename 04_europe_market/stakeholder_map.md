# Stakeholder Map

## 汽车品牌方

关注点：
- 品牌曝光质量
- 车型展示效果
- 与欧洲本地品牌形象是否匹配
- 活动数据是否可衡量
- 是否能支持多城市复制

需要提供：
- 品牌主题方案
- 车型展示配置
- 线索数据报告
- 活动照片和 PR 素材

## 进口商 / 国家销售公司

关注点：
- 上市节奏
- 经销商支持
- 预算效率
- 区域覆盖
- 合规与风险控制

需要提供：
- Benelux pilot plan
- 成本框架
- 城市和场地优先级
- Go / Conditional Go / No-Go matrix

## 经销商

关注点：
- 是否带来真实客户
- 线索归属是否清楚
- 是否需要派人
- 试驾预约是否可承接

需要提供：
- 每站线索分配规则
- Host + dealer staffing 模型
- 48 小时跟进流程
- 活动后报告模板

## 场地方

关注点：
- 安全
- 人流影响
- 安装和拆卸时间
- 保险
- 品牌调性是否匹配

需要提供：
- 结构参数
- 运输和吊装说明
- 用电需求
- 安全与保险文件
- 活动日期和动线图

## 物流与技术伙伴

关注点：
- 尺寸、重量、路线
- 吊装或放置方式
- 进出场时间
- 是否需要特殊许可

需要提供：
- box 规格
- 站点地址
- 场地进出口图
- 时间窗口
- 现场联系人

## 保险与合规方

关注点：
- 临时结构责任
- 车辆安全
- 公众责任
- 员工和访客安全
- 不同国家法规差异

需要提供：
- 结构认证
- 现场风险评估
- 运营流程
- 安全预案
- 责任边界
